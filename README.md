# 发财幸运大转盘 (Lucky Wheel)

一个以移动优先为主的前端幸运转盘示例，随机抽取 1～49 的数字。此版本重点改进了移动端显示与结果一致性：停止后的实际落位与中间显示完全一致，并且中间数字在手机上清晰可见。

主要改进
- 移动优先的响应式布局与触屏友好交互（点击画布/按钮/空格触发）。
- Canvas 高分辨率渲染，支持 Retina/高 DPI 屏幕。
- 精确对齐逻辑：计算准确的最终角度，确保停止时落在指针位置的扇区，并把结果清晰地显示在中间圈。
- 平滑减速动画（requestAnimationFrame + 缓动函数）。
- 历史记录保存在 localStorage，防重复抽取（最多保留最近 7 条）。
- 可配置的最大抽奖次数（默认 7 次），支持重置并提示操作。
- 修复原始实现里未定义 restartButton 的错误，提升可用性。

如何运行
1. 克隆仓库：
   git clone https://github.com/Leeri1y/lucky-wheel.git

2. 打开浏览器：
   直接在浏览器中打开仓库根目录的 `index.html` 文件（无需构建）。

使用说明（移动端优先）
- 点击画布或“开始”按钮，或按空格键开始抽奖。
- 中间圆圈会在停止后显示最终中奖号码（清晰、醒目）。
- 点击“重置”可以清空历史并恢复抽奖次数（会弹出确认）。
- 抽奖次数用尽时会禁用“开始”按钮，并在页面上显示提示。

后续可以做的增强
- 为转盘添加音效（转动/落定）。
- 支持自定义扇区文本（奖品名/图片）。
- 将结果上报到后端以保证公平或进行统计。
- 封装为可重用组件（React / Web Component）。
